# О программе
Данная программа позволяет использовать некоторые алгоритмы поиска в графе. Используйте как модуль. Поддерживаются ориентированный и неориентированные графы с произвольными весами.
## Требования
* `Python 3.10`
* `Jupyter notebook`
## Модули
* `Node.py` - вершина графа
* `Edge.py` - ребро графа
* `Graph.py` - граф
* `Report.ipynb` - отчет
